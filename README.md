## Teknologi yang Digunakan

Bootstrap 5 dipilih karena menyediakan sistem grid responsif, komponen UI siap pakai (navbar, card, button, form), serta dukungan aksesibilitas yang baik. Framework ini memudahkan pengembangan website yang konsisten, responsif di berbagai ukuran layar (mobile, tablet, desktop), dan sesuai dengan standar pengembangan front-end modern.

Teknologi Pendukung:
* HTML5 (struktur semantik)
* CSS3 & SCSS (custom styling)
* JavaScript (interaktivitas & simulasi data)
* Font Awesome & Bootstrap Icons (ikon UI)

---

## Fitur Utama

* Pencarian Tiket Pesawat: Form pencarian berdasarkan kota asal, kota tujuan, tanggal keberangkatan/pulang, jumlah penumpang, dan kelas penerbangan.
* Hasil Pencarian Interaktif: Menampilkan daftar tiket pesawat (data simulasi) lengkap dengan maskapai, jam penerbangan, dan harga.
* Favorit Penerbangan: Pengguna dapat menandai penerbangan favorit yang disimpan menggunakan localStorage.
* Riwayat Pencarian & Pemesanan: Menyimpan data pencarian dan pemesanan tiket secara lokal di browser.
* Destinasi Wisata Populer: Galeri destinasi wisata Indonesia yang dapat langsung digunakan untuk melakukan pencarian tiket.

---

## Struktur Halaman 

Struktur halaman website Travel.com disusun secara hierarkis sebagai berikut:
* Home / Cari Tiket (index.html)
  Halaman utama untuk pencarian tiket pesawat dan menampilkan destinasi wisata populer.
* Favorit (favorite.html)
  Menampilkan daftar penerbangan yang disimpan sebagai favorit.
* Riwayat (history.html)
  Menampilkan riwayat pencarian dan pemesanan tiket pengguna.
* Akun (account.html)
  Menampilkan informasi profil pengguna.
